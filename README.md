# Tetris — Mila Mode

Tetris limpo, sem anúncios, feito para jogar no iPhone e no desktop.

## ✦ Mila Mode

Ativa um modo especial que remove os blocos S e Z (os mais frustrantes) e deixa o jogo mais fluido e tranquilo. Usa apenas as peças I, O, T, J e L.

## Jogar online

👉 https://cercao.github.io/mila-tetris/

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy

Push para `main` dispara o deploy automático via GitHub Actions → GitHub Pages.

## Controles

**Desktop:** ← → mover | ↑ rotacionar | ↓ soft drop | Espaço hard drop | C segurar | P/Esc pausar

**Mobile:** swipe ← → mover | swipe ↓ soft drop | flick ↓ hard drop | tap rotacionar
